3.选择第二个，复制Request到脚本中替换脚本中的Request code，这时候可以关闭脚本了，然后用python运行wingide6.py


4.用python运行wingide6.py  ,把算出的Code复制到上图第二个provideed activation key here 下面的输入框中，点击continue  ，大功告成！
